
# ChemEquip Web (React)

Web client for the Chemical Equipment Parameter Visualizer.

## Run (Windows)

Prereqs:
- Node.js 18+
- Backend running (see repository root README)

Commands:
```powershell
cd web
npm install
npm run dev
```

Open: `http://localhost:5173/`

## Configure API URL

The API base URL is entered on the **Register** page (example: `http://127.0.0.1:8000/api`).

## Build

```powershell
npm run build
```
